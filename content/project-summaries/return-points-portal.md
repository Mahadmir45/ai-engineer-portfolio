# Hall 3 Return Points Portal

**Domain:** Full-stack web  
**GitHub:** Mahadmir45/hall3-return-points-portal

## Overview

Full-stack portal for tutors managing return points at Hall 3 — demonstrates production web engineering alongside AI/ML work.

## Tech stack

Next.js 15, TypeScript, Prisma, PostgreSQL, NextAuth, BullMQ, Redis, S3/R2 object storage

## Features

- Authentication and role-based access
- Background job processing with BullMQ
- File uploads to cloud storage
- Real-time status updates

## Portfolio relevance

Provides full-stack engineering credibility — shows ability to ship complete products, not just ML notebooks or scripts.
