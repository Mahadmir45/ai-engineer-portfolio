# QuantFin Quantitative Finance Library

**Domain:** Finance / quant ML  
**GitHub:** Mahadmir45/QuantFin

## Overview

Production-ready quantitative finance library unifying options pricing, portfolio optimization, algorithmic trading strategies, and risk analytics.

## Modules

- **Options:** Black-Scholes, binomial, Monte Carlo pricing; Greeks; implied vol surfaces
- **Portfolio:** Mean-variance, CVaR, risk parity optimization; VaR/drawdown analysis
- **Strategies:** Technical indicators, statistical arbitrage, ML-based (XGBoost, LSTM), topology alpha (TDA)
- **Backtesting:** Execution simulation, performance metrics, walk-forward validation
- **Research:** Factor analysis, signal generation, alpha research tools

## Topology Alpha strategy

Combines Topological Data Analysis (TDA) with Laplacian diffusion on correlation graphs to generate trading signals — bridges graph algorithms and quantitative finance.

## Tech stack

Python 3.10+, NumPy, Pandas, scikit-learn, XGBoost, PyMC, custom C++ spectral components

## Portfolio relevance

Shows depth in financial data modeling, ML strategy development, and rigorous backtesting — foundational skills for building AI systems that reason over market data.
